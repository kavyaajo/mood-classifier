# Mood Classifier 😄

A simple and fun web app built with **HTML, CSS, and JavaScript** that detects the user's mood based on their text input. It classifies input as **Happy**, **Sad**, or **Neutral** using keyword matching.

---

## 🧠 Features
- Classifies mood as:
  - 😊 Happy
  - 😔 Sad
  - 😐 Neutral
